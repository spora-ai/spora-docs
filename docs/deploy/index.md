## Deploy

Spora runs anywhere PHP 8.4+ runs. Pick the scenario that matches your environment.

- **[Docker](/deploy/docker/)** — single container, multi-container, custom builds, or prebuilt images from GHCR
- **[Shared host](/deploy/shared-host/)** — cPanel, Plesk, generic LAMP (with limitations)
- **[Classical server](/deploy/classical-server/)** — bare-metal or VPS with nginx + PHP-FPM, systemd, supervisor
- **[Local](/deploy/local/)** — laptop, Ollama, LM Studio
