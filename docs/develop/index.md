## Develop

Spora grows in two ways: **plugins** that drop into any Spora install, and **projects** that are standalone Spora-compatible applications.

- **[Plugins](/develop/plugins/)** — package tools, drivers, recipes, or app extensions as a Composer package
- **[Projects](/develop/projects/)** — scaffold your own Spora project with `spora-maker`
